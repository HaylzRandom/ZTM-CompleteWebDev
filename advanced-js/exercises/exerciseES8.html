Q3: Get the below object to go from:

				
let obj = {
  my: 'name',
  is: 'Rudolf',
  the: 'raindeer'
}
// to this:
'my name is Rudolf the raindeer'

Answer: 

                            
Object.entries(obj)
	.map((value) => value.join(' '))
	.join(' ');
