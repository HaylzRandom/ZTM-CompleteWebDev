Q1: Change this function into a ternary and assign it to variable called experiencePoints 

	
function experiencePoints() {
    if (winBattle()) {
        return 10;
    } else {
        return 1;
    } 
}

Answer: 


function experiencePoints() {
    winBattle() ? 10 : 1;
}
