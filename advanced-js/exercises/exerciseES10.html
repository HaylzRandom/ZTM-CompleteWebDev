Q1: Turn this array into a new array: [1,2,3,[4],[5]]. Bonus if you can do it on one line

const array = [[1],[2],[3],[[[4]]],[[[5]]]]

Answer: console.log(array.flat(2));

Q2: Turn this array into a new array: [ 'Hello young grasshopper!', 'you are', 'learning fast!' ]

Answer: 

                            
const greeting = [
	['Hello', 'young', 'grasshopper!'],
	['you', 'are'],
	['learning', 'fast!'],
];

console.log(greeting.flatMap((greet) => greet.join(' ')));

Q3: Turn the greeting array above into a string: 'Hello young grasshopper you are learning fast!'

Answer:
console.log(greeting.flatMap((greet) => greet.join(' ')).join(' '));

const trapped = [[[[[[[[[[[[[[[[[[[[[[[[[[3]]]]]]]]]]]]]]]]]]]]]]]]]];

Q6: Turn the below users (value is their ID number) into an array: [ [ 'user1', 18273 ], [ 'user2', 92833 ], [ 'user3', 90315 ] ]

Answer: 

                            
const users = { user1: 18273, user2: 92833, user3: 90315 }
const usersArray = Object.entries(users);

Q7: Change the output array of the above to have the user's IDs multiplied by 2 -- Should output:[ [ 'user1', 36546 ], [ 'user2', 185666 ], [ 'user3', 180630 ] ]

Answer: const updatedUsersArray = usersArray.map((user) => [user[0], user[1] * 2]);

Q8: Change the output array of question #7 back into an object with all the users IDs updated to their new version. Should output: { user1: 36546, user2: 185666, user3: 180630 }

Answer: 

                            
const updatedUsers = Object.fromEntries(updatedUsersArray);
console.log(updatedUsers);
