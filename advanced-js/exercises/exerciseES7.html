Advanced JavaScript Exercises - Exercise ES7

Question: Solve the below problems

Q1: Check if this array includes the name "John"

const dragons = ['Tim', 'Johnathan', 'Sandy', 'Sarah'];

Answer: dragons.includes('John'); // Returns false

Q2: Check if this array includes any name that has "John" inside of it. If it does, return that // name or names in an array.

const dragons = ['Tim', 'Johnathan', 'Sandy', 'Sarah'];

Answer: dragons.filter((name) => name.includes('John')); // Returns 'Johnathan'

Q3: Create a function that calculates the power of 100 of a number entered as a parameter

Answer: const power100 = (number) => number ** 100;

Q4: Using your function from #3, put in the parameter 10000. What is the result?

Answer: power100(10000); // Infinity
