Advanced JavaScript Exercises - Exercise 4

Question: Solve these problems

Q1: Create a one line function that adds adds two parameters

Answer: const add = (a, b) => a + b;

Q2: Closure: What does the last line return?

				

const addTo = x => y => x + y
var addToTen = addTo(10)
addToTen(3)

Answer: The value returned will be 13

Q3: Currying: What does the last line return?

				
const sum = (a, b) => a + b
const curriedSum = (a) => (b) => a + b
curriedSum(30)(1)

Answer: The value returned will be 31

Q4: Currying: What does the last line return?

				
const sum = (a, b) => a + b
const curriedSum = (a) => (b) => a + b
const add5 = curriedSum(5)
add5(12)

Answer: The value returned will be 17

Q5: Composing: What does the last line return?

				
const compose = (f, g) => (a) => f(g(a));
const add1 = (num) => num + 1;
const add5 = (num) => num + 5;
compose(add1, add5)(10)

Answer: The value returned will be 16

Q6: What are the two elements of a pure function?

Answer:

  • Determenistic - Always produces same results when given same inputs
  • No Side Effects - Doesn't depend on any state, data or change during a program's execution. Must only depend on input elements
